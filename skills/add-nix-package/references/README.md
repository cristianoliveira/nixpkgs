# Stack Reference Index

Use these short references when adding a package under `pkgs/<name>/default.nix`.

- `go.md`: Go CLIs and services built from source.
- `rust.md`: Rust binaries using Cargo.
- `node.md`: Node packages and web assets with npm/yarn.
- `python.md`: Python CLI/apps/modules from PyPI or source.
- `binary-release.md`: Prebuilt release archives with platform-specific sources.

Each file includes:

- when to use the pattern
- common fields and helpers
- a minimal snippet
- concrete `.local/nixpkgs/*` examples
- stack-specific pitfalls and checks
